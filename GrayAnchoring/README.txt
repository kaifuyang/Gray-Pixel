INTRODUCTION
===================================================================================
This code is provided for research purpose only.
This code performs Gray Anchoring reported in:

Kai-Fu Yang, Dajun Xing, Yong-Jie Li. 
Gray Anchoring: a New Computational Theory for Biological Color Constancy. Neuroscience Bulletin, 2026.


Please cite our paper if this code is used to motivate any publications.
